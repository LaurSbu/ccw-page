TheApp.vip — Features & Dashboards

TheApp.vip is a creator-first platform for live streaming engagement, donations and overlays. Below are the main features and how to use them.

TheApp Dashboard

The central hub where you connect your platforms and review analytics. From here you can manage connections to TikTok, YouTube and Kick, and view performance across time ranges (last 7 days, last month, last year).

Analytics text: You can view analytics for your live sessions, including statistics on likes, comments, shares, views, and donations. Additionally, there's a separate chart for donations categorized by link.

TikTok Dashboard — Alerts, Goals, Chatbots, Trolls, Tops

Complete control over TikTok live overlays and behaviors. All alerts provide an OBS integration URL for use as a browser source.

  • Alerts (Member, Gift, Like, Emote, Envelope, Follow, Share, Subscribe): rename alerts, choose designs, set priority (0–3), toggle overwrite-of-lower-priority behavior, restrict triggers (everyone, followers, subscribers, moderators, new gifters, top gifters, specific users), and set min/max activation values where applicable (e.g. gift coin minimums, like min/max).
  • Goals: choose data source (current live or all-time/general stats), select metric (Likes, Donations, Shares, Comments), set target value, and pick a design for the visual goal bar.
  • Chat Bots (TTS): configure trigger prefix (e.g. "/"), language, gender, speed, pitch, allowed triggerers, and optional point cost if Points System enabled.
  • Trolls: let viewers trigger keypresses, mouse moves/clicks or sequences (cost via gifts or system points), configure durations (ms), multi-key behavior and designs for the overlay explanation.
  • Tops: leaderboards for most donations/likes/shares/comments; select number of top users and data source.

Donations Dashboard

TheApp.vip donation engine is designed to maximize donations and let you customize how donations impact your stream (alerts, tops, goals, trolls, history).

Alerts

Donation alerts are highly configurable: name, design, priority (0–3), overwrite behavior, min/max activation ranges and optional TTS with voice/gender selection.

Goals

Set date ranges and target values to display donation progress in real time.

Trolls

Donation-triggered trolls work like TikTok trolls but are activated by payments through TheApp donation system. Configure cost, key sequences, durations and designs.

Tops & History

Create donation leaderboards, set how many top users to display, and review donation history (who donated and how much).

Points System

Encourage engagement with a points economy — viewers can spend coins to receive points and redeem them on features such as Trolls or Chatbots.

  • Configure how many coins viewers must spend and the reward in points.
  • Set the platform scope (currently TikTok, more platforms coming).
  • Use points to gate interactive features and special overlays.

Design & Gallery

The design module is a powerful visual editor for building alert overlays and animations. You can combine images, timelines and tags to create complex animations.

  • Use tags (placeholders) such as {sender} or {message} in designs to show dynamic info (e.g. "Jay sent Heart").
  • Open the Info modal (top-right) to see available tags and examples.
  • Upload custom assets in Gallery and reuse them across designs.

Gallery

Upload images, GIFs, audio and videos to be used by designs and alerts. Assets are available in the design editor and can be used as overlays in OBS.

Chat

Unified chat for TikTok, YouTube and Kick. Merge gifts from multiple platforms into a single view, customize layout and visual style.

  • Adjust proportions between messages and gifts with a slider.
  • Display as row or column, change font, size, letter spacing and background opacity.
  • Color-code message backgrounds to differentiate subscribers, paid subscribers and non-subscribers.

Groups

Create link groups to combine multiple alerts and trolls into one OBS link. Useful for loading many overlays with a single browser source.

Referrals

Get a unique referral link to invite others. See monthly earnings and the last 5 months of referral performance.

Profile & Settings

Change your donation nickname, manage account settings and configure platform-specific options in the settings submenu.
